# HAR URL Extractor

Universal Python script to extract and categorise all URLs from a HAR file.

## Features
- Extracts all request URLs from HAR
- Categorises by type (JS, CSS, Images, APIs, HTML, Other)
- No scope assumptions
- Outputs Excel (.xlsx) file
- Works for any domain

## Usage
```bash
python extract_HAR_urls.py <file.har>
